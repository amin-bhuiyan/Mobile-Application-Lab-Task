## 📌 Project Overview

This project is a Flutter-based form application developed as part of the Mobile App Development Lab. The app demonstrates how to create a user input form, collect data, and display the submitted information.

---

## ✨ Features

* User-friendly form interface
* Input fields for user data
* Form validation
* Display submitted data on screen

---

## 🛠️ Technologies Used

* Flutter
* Dart

## 🚀 How to Run the Project

1. Clone the repository:
https://github.com/amin-bhuiyan/Mobile-Application-Lab-Task/tree/Form-creation/Form%20Creation/lib
2. Navigate to the project folder:

   cd form_practice_01
   
3. Get dependencies:

   
   flutter pub get
   
4. Run the app:

   
   flutter run
